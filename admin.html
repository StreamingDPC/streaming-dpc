Guía para Gmail

Para usar Gmail, debes generar una Contraseña de Aplicación:

  1. Activa la "Verificación en 2 pasos" en tu cuenta Google.
  2. Busca "Contraseñas de aplicaciones" en tu perfil.
  3. Genera una para "Correo" y copia el código de 16 letras.
  4. Usa ese código de 16 letras como contraseña aquí abajo.
